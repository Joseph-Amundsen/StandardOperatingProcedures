<h1>Setting up 2-Step Verification for a Google Account</h1>

<h2>Description</h2>
This project provides a Standard Operating Procedure (SOP) for guiding individuals through the process of enabling 2-Step Verification (2SV) on their Google account. The document is designed to help users improve their account security by adding an additional authentication step beyond a password.
<br />


<h2>Objectives</h2>

- <b>Provide clear, step-by-step instructions for enabling Google 2-Step Verification.</b> 
- <b>Help users understand the importance of multi-factor authentication.</b>
- <b>Reduce the risk of unauthorized access to Google accounts.</b> 
- <b>Create a standardized process that can be used for training, onboarding, or IT support.</b>

<h2>Intended Audience </h2>

- <b>End Users</b>
- <b>IT support staff</b>
- <b>Help desk technicians</b>
- <b>Trainers and onboarding personnel</b>
